1 Program Nedir? GUI vs CLI

🖱️ GUI

Grafik Arayüz — fare ile tıklanan, görsel programlar

Firefox, Photoshop, Word, mobil uygulamalar

⌨️ CLI

Komut Satırı — metin tabanlı, komut yazılan programlar

Terminal, PowerShell, komut yazarak çalıştırma

CLI Neden Önemli?

  • Hız: Deneyimli kullanıcılar için çok daha hızlı
  • Otomasyon: Komutları zincirleme çalıştırabilirsiniz (scripting)
  • Sunucularda zorunluluk: Uzak sunucularda GUI yoktur
  • AI ile kolay: AI metin tabanlı program yazmakta çok daha başarılı
📝 Not: Bu derslerde yazacağımız programlar CLI tabanlı olacak.

⌨️ Terminalde Kullanılan CLI Programları

KomutAçıklamaÖrnek
dateSistem tarih ve saatini gösterirdate +"%Y-%m-%d"
calTerminal içinde takvim görüntülercal 2026
topCanlı sistem kaynak kullanımını izler (CPU, RAM)top veya htop
wgetWeb'den dosya indirirwget https://example.com/data.csv
ollamaYerel bilgisayarda LLM çalıştırırollama run llama3
💡 Bu komutların hepsi GUI'siz çalışır: ekranda pencere açmaz, metin girer / metin çıkarır. Terminal'e yazıp Enter'a basın.

and if it works — great."

— Andrej Karpathy, 2025

⚠️ "Vibe Coding" Eleştirisi

  • Güvenlik açıkları: İncelenmemiş kod, sızıntı riski

💡 Açık vs Kapalı: Kapalı kaynak = şirket sunucuları. Açık kaynak = kendi sunucunuzda çalıştırılabilir.

🔤 Token Kavramı Tokens & Context Window

  • Token ≈ ¾ kelime veya 3–4 karakter
  • Context window = tek sohbetteki toplam girdi + çıktı token limiti
  • 128K ≈ 300 sayfa, 1M ≈ 2.500 sayfa
🔗 OpenAI Token Calculator
⚠️ Token Limiti Neden Önemli?
  • Büyük kod tabanları tek sohbete sığmaz
  • Sohbet uzadıkça halüsinasyon (uydurma) artar, geçmiş unutulur
  • API'de her token ücretli

→ Cursor, Windsurf gibi AI IDE'ler tüm codebase yerine sadece ilgili dosyaları akıllıca seçerek LLM'e gönderir.

10 Bu Yöntemin Sınırları Limitations

  • Context Switch: ChatGPT ↔ IDE arası sürekli gidip gelme
  • Farklı LLM'ler farklı alanlarda iyi: En iyisini seçmek zor
  • Context window limiti: Tüm projeyi gönderemezsiniz
  • Otomasyon yok: ChatGPT kodu yazar ama çalıştırmaz, hata ayıklamaz
→ Bu sınırlar, doğrudan IDE'ye entegre AI kodlama araçlarının doğmasına yol açtı.

14 Teknik Bilgi Neden Hâlâ Önemli? Why It Matters

✈️ Pilot ve Otopilot Analojisi

  • Modern uçaklarda gelişmiş otopilot var — ama pilotlar hâlâ gerekli
  • Beklenmedik durumlar insan kararı gerektirir
  • Pilot doğru parametreleri girer, otopilotun ne yaptığını anlar
AI kodlama araçları = otopilot, siz = pilot

Teknik Bilginin Somut Avantajları

  • Daha iyi prompt: "responsive design, grid layout" vs "telefonda güzel görünsün"
  • Doğru yönlendirme: "cache ekle", "binary search kullan", "async/await yap"
  • Doğru kütüphane seçimi: Hangi araç ne işe yarar bilmek
  • Hataları anlama: AI'ın hatasını yakalamak
  • Sistem tasarımı: HTTP, API, REST, DNS, client/server kavramları
🌐 UI bileşen isimleri: component.gallery
"Accordion", "modal", "tabs", "breadcrumbs" gibi terimleri bilmek AI ile iletişimi güçlendirir.
